# 首页

## 如何参与

1. 阅读 [贡献指南](/contributing) 了解流程。
2. 如果你是首次投稿者，请先提交申请 Issue。
3. 申请通过并加入协作者后，后续文章可直接分支 + PR。

申请入口：
<https://github.com/PerAsperaFTAJ/PerAsperaFTAJ.github.io/issues/new?template=wiki-change-request.yml>
